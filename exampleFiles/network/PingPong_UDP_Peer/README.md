# UDP Peer-to-Peer Ping-Pong

This setup uses two Docklight peer projects with cross-connected localhost ports for UDP message exchange. It demonstrates mirrored configuration for dual-instance peer communication.

## Requirements

- Docklight Scripting (network example: UDP peer mode)

## Files

| File | Description |
|------|-------------|
| PingPong_UDP_Peer1.ptn | Project documentation (Notepad) |
| PingPong_UDP_Peer1.ptp | Docklight project file |
| PingPong_UDP_Peer1_ptn.txt | Text export of documentation |
| PingPong_UDP_Peer1_ptp.txt | Text export of project file |
| PingPong_UDP_Peer2.ptp | Docklight project file |
| PingPong_UDP_Peer2_ptp.txt | Text export of project file |

## Getting Started

1. Open `PingPong_UDP_Peer1.ptp` in one Docklight Scripting instance.
2. Open `PingPong_UDP_Peer2.ptp` in a second Docklight Scripting instance.
3. Press `F5` in both instances, then send ping messages to verify peer-to-peer exchange.
