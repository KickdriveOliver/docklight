# UDP Loopback Ping-Pong

This is the simplest UDP example and runs in a single Docklight instance using localhost loopback. It is intended as a quick smoke test for UDP send/receive configuration.

## Requirements

- Docklight Scripting (network example: UDP)

## Files

| File | Description |
|------|-------------|
| PingPong_UDP_Loopback.ptn | Project documentation (Notepad) |
| PingPong_UDP_Loopback.ptp | Docklight project file |
| PingPong_UDP_Loopback_ptn.txt | Text export of documentation |
| PingPong_UDP_Loopback_ptp.txt | Text export of project file |

## Getting Started

1. Open `PingPong_UDP_Loopback.ptp` in Docklight Scripting.
2. Keep the default localhost UDP settings and press `F5`.
3. Send the predefined ping sequence to verify loopback receive/transmit in one instance.
