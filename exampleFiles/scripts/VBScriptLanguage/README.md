# VBScript Language Reference for Docklight

This directory is a compact VBScript language reference tailored for Docklight Scripting users. Each category contains focused `.pts` examples (plus matching `_pts.vbs` exports) that demonstrate a specific language element you can run and adapt immediately.

## Requirements

- Docklight Scripting

## Files

| File | Description |
|------|-------------|
| Functions/ | Subdirectory with additional examples |
| Methods/ | Subdirectory with additional examples |
| Objects/ | Subdirectory with additional examples |
| Properties/ | Subdirectory with additional examples |
| Statements/ | Subdirectory with additional examples |

## Getting Started

1. Open any `.pts` example from the `Functions`, `Methods`, `Objects`, `Properties`, or `Statements` subfolders in Docklight Scripting.
2. Press `Shift+F5` to run the script example and inspect the corresponding behavior/output.
