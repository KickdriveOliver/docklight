# Database ODBC Connection

This example connects Docklight Scripting to an ODBC data source (Access sample included) for database-backed automation. It ships with helper script code, setup docs, and screenshots for ODBC configuration.

## Requirements

- Docklight Scripting (script-based workflow)

## Getting Started

1. Read `Docklight_ODBC_example.pdf` (or `.docx`) first for ODBC DSN setup prerequisites.
2. Open `Docklight_ODBC_example.pts` in Docklight Scripting.
3. Start the script with `Shift+F5`.
