# Float Conversion Example

This script-only reference demonstrates IEEE 754 single and double precision conversion utilities for protocol payload decoding. It is intended as a reusable helper pattern for scripting workflows.

## Requirements

- Docklight Scripting (script-based workflow)

## Files

| File | Description |
|------|-------------|
| FloatConversionExample.pts | Docklight Script |
| FloatConversionExample_pts.vbs | VBScript export of script |

## Getting Started

1. Open `FloatConversionExample.pts` in Docklight Scripting.
2. Run the script with `Shift+F5`.
