# Close and Reopen Delay

This script-only example demonstrates controlled serial port close/reopen cycles with configurable delay intervals. It is useful for testing reconnect behavior in unstable or reset-prone devices.

## Requirements

- Docklight Scripting (script-based workflow)

## Files

| File | Description |
|------|-------------|
| close_and_reopen_delay.pts | Docklight Script |
| close_and_reopen_delay_pts.vbs | VBScript export of script |

## Getting Started

1. Open `close_and_reopen_delay.pts` in Docklight Scripting.
2. Run the script with `Shift+F5`.
