# Docklight Excel Import/Export

This folder contains Excel VBA tooling for importing/exporting Docklight project data. It includes macro-enabled workbook assets, VBA exports, and the ExampleProjects subfolder with sample project files.

## Requirements

- Docklight
- Microsoft Excel (macro-enabled workbook support)

## Files

| File | Description |
|------|-------------|
| ExampleProjects/ | Subdirectory with additional examples |
| Docklight_Excel_Import_Export_Code_and_Examples.txt | Text export or notes |
| Docklight_Excel_Import_Export_VBA_Code.bas.txt | Text export or notes |
| DocklightImportExport.xlsm | Excel workbook with VBA |
| DocklightImportExport_Example_Screenshot.png | Screenshot |
| DocklightImportExport_Example_Screenshot_2.png | Screenshot |

## Getting Started

1. Open `Docklight_Excel_Import_Export_Code_and_Examples.txt` for usage notes and sample workflows.
2. Open `DocklightImportExport.xlsm` in Excel and enable macros.
3. Use the VBA import/export functions with the sample projects in `ExampleProjects/`.
