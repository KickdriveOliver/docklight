# Send Byte Timing

This example configures inter-character timing delays within send sequences to mimic slower or timing-sensitive transmitters. Screenshots in the folder illustrate timing behavior.

## Requirements

- Docklight Scripting (script-based workflow)
- Docklight can be used for project-only runs where no script is required

## Files

| File | Description |
|------|-------------|
| readme.txt | Text export or notes |
| SendByteTiming.pts | Docklight Script |
| SendByteTiming_pts.vbs | VBScript export of script |
| SendByteTimingTest.ptp | Docklight project file |
| SendByteTimingTest_ptp.txt | Text export of project file |
| SendByteTimingTest_Screenshot1.png | Screenshot |
| SendByteTimingTest_Screenshot2.png | Screenshot |

## Getting Started

1. Open `SendByteTimingTest.ptp` in Docklight Scripting.
2. Configure the interface parameters for your setup and press `F5` to start communication.
3. If you want the script-driven variant, open `SendByteTiming.pts` in Docklight Scripting and start it with `Shift+F5`.
