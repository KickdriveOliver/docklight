# Parity Switch for 9-Bit Protocols

This demo switches serial parity settings on the fly to emulate 9-bit master/slave addressing workflows. It is relevant for multidrop and protocol variants that embed address/data distinction in parity.

## Requirements

- Docklight

## Files

| File | Description |
|------|-------------|
| Docklight_SwitchParityDemo.ptn | Project documentation (Notepad) |
| Docklight_SwitchParityDemo.ptp | Docklight project file |
| Docklight_SwitchParityDemo_ptn.txt | Text export of documentation |
| Docklight_SwitchParityDemo_ptp.txt | Text export of project file |

## Getting Started

1. Open `Docklight_SwitchParityDemo.ptp` in Docklight.
2. Configure the interface parameters for your setup and press `F5` to start communication.
