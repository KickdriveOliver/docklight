# Modbus RTU CRC

This example illustrates Modbus RTU frame handling with CRC calculation and decoding. It can be studied with loopback setup even when no Modbus hardware is connected.

## Requirements

- Docklight

## Files

| File | Description |
|------|-------------|
| ModbusRtuCrc.ptn | Project documentation (Notepad) |
| ModbusRtuCrc.ptp | Docklight project file |
| ModbusRtuCrc_ptn.txt | Text export of documentation |
| ModbusRtuCrc_ptp.txt | Text export of project file |

## Getting Started

1. Open `ModbusRtuCrc.ptp` in Docklight.
2. Configure the interface parameters for your setup and press `F5` to start communication.
