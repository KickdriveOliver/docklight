# Gardasoft RT LED Controller

This project documents communication with Gardasoft Vision RT series LED lighting controllers. It is focused on straightforward serial command testing with matching project notes.

## Requirements

- Docklight
- Gardasoft RT controller

## Files

| File | Description |
|------|-------------|
| Gardasoft_RT.ptn | Project documentation (Notepad) |
| Gardasoft_RT.ptp | Docklight project file |
| Gardasoft_RT_ptn.txt | Text export of documentation |
| Gardasoft_RT_ptp.txt | Text export of project file |

## Getting Started

1. Open `Gardasoft_RT.ptp` in Docklight.
2. Configure the interface parameters for your setup and press `F5` to start communication.
