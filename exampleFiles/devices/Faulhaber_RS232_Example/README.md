# Faulhaber RS232 Motion Controller

This example demonstrates serial command/response communication with Faulhaber motion controllers from the MCBL and MCDC families. It provides a compact project + documentation pair suitable for first-connect validation.

## Requirements

- Docklight
- Faulhaber controller connected via RS232

## Files

| File | Description |
|------|-------------|
| Faulhaber_RS232.ptn | Project documentation (Notepad) |
| Faulhaber_RS232.ptp | Docklight project file |
| Faulhaber_RS232_ptn.txt | Text export of documentation |
| Faulhaber_RS232_ptp.txt | Text export of project file |

## Getting Started

1. Open `Faulhaber_RS232.ptp` in Docklight.
2. Configure the interface parameters for your setup and press `F5` to start communication.
