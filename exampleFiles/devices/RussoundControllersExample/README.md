# Russound RS-232 Controller Integration

This folder contains RS-232 communication setups for Russound multi-zone audio controllers. It combines project text exports and product-specific documentation for practical controller integration.

## Requirements

- Docklight
- Russound controller with RS-232 interface

## Files

| File | Description |
|------|-------------|
| RUSSOUND RS-232 DOCKLIGHT.ptp | Docklight project file |
| RUSSOUND RS-232 DOCKLIGHT_ptp.txt | Text export of project file |
| using docklight with russound controller.pdf | Documentation |

## Getting Started

1. Read `using docklight with russound controller.pdf` first for controller model specifics and command usage.
2. Open `RUSSOUND RS-232 DOCKLIGHT.ptp` in Docklight.
3. Configure the interface parameters for your setup and press `F5` to start communication.
