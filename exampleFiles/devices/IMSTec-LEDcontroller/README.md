# IMSTec LED Controller

This setup targets the IMSTec Dual Intelligent Constant Current LED Controller (ILC2008), including a 350mA control script variant. It provides project and script assets for quick command testing and repeatable operation.

## Requirements

- Docklight Scripting (script-based workflow)
- Docklight can be used for project-only runs where no script is required
- IMSTec ILC2008 LED controller

## Files

| File | Description |
|------|-------------|
| 350mA-Example.pts | Docklight Script |
| 350mA-Example_pts.vbs | VBScript export of script |
| IMSTec-LEDcontroller.ptn | Project documentation (Notepad) |
| IMSTec-LEDcontroller.ptp | Docklight project file |
| IMSTec-LEDcontroller_ptn.txt | Text export of documentation |
| IMSTec-LEDcontroller_ptp.txt | Text export of project file |

## Getting Started

1. Open `IMSTec-LEDcontroller.ptp` in Docklight Scripting.
2. Configure the interface parameters for your setup and press `F5` to start communication.
3. If you want the script-driven variant, open `350mA-Example.pts` in Docklight Scripting and start it with `Shift+F5`.
