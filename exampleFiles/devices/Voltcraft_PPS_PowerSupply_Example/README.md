# Voltcraft PPS Power Supply

This example targets serial control of Voltcraft PPS programmable power supplies. It provides a minimal but practical setup for command and response testing.

## Requirements

- Docklight
- Voltcraft PPS series programmable power supply

## Files

| File | Description |
|------|-------------|
| Voltcraft_PPS_PowerSupply.ptn | Project documentation (Notepad) |
| Voltcraft_PPS_PowerSupply.ptp | Docklight project file |
| Voltcraft_PPS_PowerSupply_ptn.txt | Text export of documentation |
| Voltcraft_PPS_PowerSupply_ptp.txt | Text export of project file |

## Getting Started

1. Open `Voltcraft_PPS_PowerSupply.ptp` in Docklight.
2. Configure the interface parameters for your setup and press `F5` to start communication.
