# Logitech Wireless Rumblepad 2 (USB HID)

This example captures and interprets USB HID events from a Logitech Wireless Rumblepad 2 gamepad. It demonstrates Docklight HID communication mode for non-serial device traffic.

## Requirements

- Docklight Scripting (USB HID channel)
- Logitech Wireless Rumblepad 2

## Files

| File | Description |
|------|-------------|
| LogitechWirelessRumblepad2Example.ptn | Project documentation (Notepad) |
| LogitechWirelessRumblepad2Example.ptp | Docklight project file |
| LogitechWirelessRumblepad2Example_ptn.txt | Text export of documentation |
| LogitechWirelessRumblepad2Example_ptp.txt | Text export of project file |

## Getting Started

1. Open `LogitechWirelessRumblepad2Example.ptp` in Docklight Scripting.
2. Configure the interface parameters for your setup and press `F5` to start communication.
