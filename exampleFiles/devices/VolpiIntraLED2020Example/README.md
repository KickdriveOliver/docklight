# Volpi IntraLED 2020

This example uses the Volpi IntraLED 2020 ASCII control protocol for LED light source configuration. It includes both project and scripting variants for interactive and automated runs.

## Requirements

- Docklight Scripting (script-based workflow)
- Docklight can be used for project-only runs where no script is required
- Volpi IntraLED 2020

## Files

| File | Description |
|------|-------------|
| VolpiIntraLED2020.ptn | Project documentation (Notepad) |
| VolpiIntraLED2020.ptp | Docklight project file |
| VolpiIntraLED2020.pts | Docklight Script |
| VolpiIntraLED2020_ptn.txt | Text export of documentation |
| VolpiIntraLED2020_ptp.txt | Text export of project file |
| VolpiIntraLED2020_pts.vbs | VBScript export of script |

## Getting Started

1. Open `VolpiIntraLED2020.pts` in Docklight Scripting.
2. Press `Shift+F5` to start the script. It loads the project via `DL.OpenProject` (target: `VolpiIntraLED2020.ptp`).
