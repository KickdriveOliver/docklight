\par \cf1\b0 This script shows the current state of modes and values of the UNI-T UT803 inside
\par the Docklight output tab.
\par 
\par You can start/stop a data export/logging in csv format using the [1] and [0] keys. 
\par 
\par TIP: Afterwards you can open this file in Excel to create a diagram for further visualization.
\par 
\par \cf0 Precondition: 
\par - RS232 Cable connected
\par - \cf1 UNI-T UT803 powered\cf0 
\par - adjust the COM port no. according to your system environment
\par - Script started [Shift+F5]\cf1
