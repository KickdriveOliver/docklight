# UNI-T UT803 Digital Multimeter

This example demonstrates live acquisition and display from a UNI-T UT803 digital multimeter, including CSV export workflows. It includes main and tester projects, scripting assets, and display templates.

## Requirements

- Docklight Scripting (script-based workflow)
- Docklight can be used for project-only runs where no script is required
- UNI-T UT803 multimeter
- Serial connection

## Getting Started

1. Open `UT803 digital multimeter.ptp` in Docklight Scripting.
2. Press `F5`. The matching `UT803 digital multimeter_auto.pts` script is loaded automatically and started together with communication.
