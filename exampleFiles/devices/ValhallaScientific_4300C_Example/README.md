# Valhalla Scientific 4300C

This project demonstrates RS232 communication with the Valhalla Scientific 4300C Digital Micro-Ohmmeter. It is a concise project/notepad example focused on command verification.

## Requirements

- Docklight
- Valhalla Scientific 4300C instrument

## Files

| File | Description |
|------|-------------|
| ValhallaScientific_4300C_Example.ptn | Project documentation (Notepad) |
| ValhallaScientific_4300C_Example.ptp | Docklight project file |
| ValhallaScientific_4300C_Example_ptn.txt | Text export of documentation |
| ValhallaScientific_4300C_Example_ptp.txt | Text export of project file |

## Getting Started

1. Open `ValhallaScientific_4300C_Example.ptp` in Docklight.
2. Configure the interface parameters for your setup and press `F5` to start communication.
