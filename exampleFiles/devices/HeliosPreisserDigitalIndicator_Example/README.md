# Helios Preisser Digital Indicator

This example reads measurement data from Helios Preisser digital indicator instruments over RS232. It is useful for validating framing and value parsing for distance/position data streams.

## Requirements

- Docklight
- Helios Preisser measuring instrument with RS232

## Files

| File | Description |
|------|-------------|
| HeliosPreisserDigitalIndicator_Example.ptn | Project documentation (Notepad) |
| HeliosPreisserDigitalIndicator_Example.ptp | Docklight project file |
| HeliosPreisserDigitalIndicator_Example_ptn.txt | Text export of documentation |
| HeliosPreisserDigitalIndicator_Example_ptp.txt | Text export of project file |

## Getting Started

1. Open `HeliosPreisserDigitalIndicator_Example.ptp` in Docklight.
2. Configure the interface parameters for your setup and press `F5` to start communication.
