# Barionet Controller Protocol Examples

These projects cover Barix Barionet controller communication for ASCII TCP, ASCII UDP, Modbus TCP, and serial gateway scenarios. Each protocol variant has its own project and notepad documentation for quick setup.

## Requirements

- Docklight Scripting
- Barix Barionet hardware

## Getting Started

1. Open the project that matches your target protocol in Docklight Scripting:
	- `Barionet_ASCII_TCP.ptp` for ASCII over TCP
	- `Barionet_ASCII_UDP.ptp` for ASCII over UDP
	- `Barionet_Modbus_TCP.ptp` for Modbus/TCP
2. Press `F5` to start communication and verify traffic.
3. Use `Barionet_Serial_Gateway.ptp` only for the serial-gateway scenario.
