\par Note: \tab\b0 If you require different Network settings, you have to change the Docklight
\par \tab Project settings. (Menu Tools / Project Settings.. / Tab Communication)
\par \tab Port 502 is registered for MODBUS commands.
\par \b 
\par Docklight Send Sequences:
\par \b0 On the left side you can find some commands according to the MODBUS protocol
\par and the Barionet I/O Adress map.
\par 15 01 00 00 00 06 : is an arbitrary MBAP Header
\par Byte 1+2: Transaction Indentifier
\par Byte 3+4 = 00 00 : MODBUS Protocol
\par Byte 5+6: Number of following bytes
\par 
\par Byte 7 : Unit identifier
\par 
\par If you want to edit a certain command just double-click on the entry
\par in the Send Sequence list.
\par 
\par \b Docklight Receive Sequences:
\par \b0 Docklight can detect certain incoming serial data with the help of definded receive
\par sequences. As an example some Barionet answers will be detected and Docklight
\par inserts additional comments in the communication window.
